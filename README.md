This project is a basic Python program that allows users to generate multiplication tables for a given number. The user inputs a number, and the program prints its table in a clean and readable format.

It is designed for beginners who are learning Python fundamentals such as loops, conditional statements, and user input handling. This project helps in understanding how to structure simple programs and improve logical thinking.

🚀 Features
Generate multiplication tables for any number
Clean and easy-to-read output
Beginner-friendly code
Uses basic Python concepts (loops, input, print)
🛠️ Technologies Used
Python 3
💡 Future Improvements (optional but impressive 🔥)
Add range support (e.g., tables from 1 to 10)
Save output to a file
Create GUI version using Tkinter
